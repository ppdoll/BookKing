1BookKing이란?

BookKing은 그룹 단위 독서 기록 서비스예요. 가족·친구·동료끼리 그룹을 만들어 읽은 책을 기록하고, 별점과 기억에 남는 문장을 나누고, 누가 제일 많이 읽었는지 랭킹으로 겨뤄요.

🙂 사용자책 등록·수정·삭제, 내 책장, 검색, 랭킹 보기, 새 그룹 만들기
🛡 운영자사용자 기능 + 그룹 글 관리(검색·수정·삭제)
👑 그룹장운영자 기능 + 운영자 지정, 초대 링크 관리, 그룹장 위임, 삭제 글 복구

※ 역할은 그룹마다 따로예요. A그룹에서는 그룹장, B그룹에서는 사용자일 수 있어요. 기록도 그룹별로 분리되어 내가 A그룹에 등록한 책은 A그룹원에게만 보여요.

2시작하기 — 가입과 그룹

초대 링크로 가입하기 (일반 멤버)

STEP 1🔗 초대 링크 열기그룹장에게 받은 링크 접속 (링크는 발급 후 7일간 유효)
→
STEP 2🔑 Google 로그인구글 계정 하나면 끝. 별도 회원가입 없음
→
STEP 3🐿️ 이름 정하기그룹원들에게 보이는 이름 등록 (필수)
→
STEP 4🎉 가입 완료홈 화면으로 이동, 바로 사용 시작
📗 가입하면 “나만의 책장”이 자동으로 생겨요 가입을 마치면 『내이름의 책장』이라는 개인 공간이 자동으로 만들어져요. 그룹에 들어가지 않아도 여기서 바로 책을 기록할 수 있어요. 개인 책장은 나만 볼 수 있고, 이용권도 쓰지 않아요.
⏰ 링크가 만료됐다고 나와요 초대 링크는 7일이 지나면 만료돼요. 그룹장에게 새 링크를 요청하세요. (그룹장은 👑 그룹 관리 → 새 링크 발급)

공개 그룹 찾아서 가입하기

초대 링크가 없어도 👥 그룹 찾기에서 검색을 허용한 공개 그룹을 찾아 가입하기 한 번으로 들어갈 수 있어요.

※ 그룹 찾기·그룹 만들기·이용권 메뉴는 내 개인 책장을 보고 있을 때 상단에 나타나요. 공유 그룹에서는 그룹 활동 메뉴만 보여서 화면이 깔끔해요. 상단의 그룹 선택에서 『내이름의 책장』으로 바꾸면 돼요.

그룹 직접 만들기

  1. 개인 책장에서 상단 메뉴 🌱 그룹 만들기 클릭
  2. 그룹 이름 입력 → 그룹 만들기 🎉
  3. 만든 사람이 자동으로 👑 그룹장이 되고, 초대 링크가 발급돼요 → 복사해서 멤버들에게 공유
🎟️ 그룹은 기본 1개까지 무료로 만들 수 있어요 초대받아 가입하는 건 무제한이지만, 그룹을 직접 만드는 것은 이용권이 필요해요 (기본 1개 제공). 더 만들고 싶으면 개인 책장의 상단 메뉴 🎟️ 이용권에서 쿠폰 코드를 입력하거나, 운영자에게 이용권을 요청해주세요 — 운영자가 승인하면 같은 페이지에서 쿠폰을 받아 바로 사용할 수 있어요.

3홈 화면 한눈에 보기

위치내용
상단바그룹 선택(여러 그룹 가입 시 전환), 메뉴, 내 이름·역할·UUID, 로그아웃
독서 보드🌱 읽을 예정 → 📖 독서중 → 🏆 완독 3단계로 내 책이 정리돼요
올해의 독서 기록월별 완독 수 막대 차트 — 올해 얼마나 읽었는지 한눈에
그룹 최근 기록그룹원들의 최신 소식. 클릭하면 그 사람의 문장·느낀 점을 볼 수 있어요
우측 랭킹책 랭킹 + 👑 독서왕 랭킹 (8장 참고)

4책 등록하기

  1. 상단 메뉴 ✏️ 책 등록 클릭
  2. 등록할 그룹 선택 — 내가 기록할 수 있는 그룹이 나열돼요. 여러 그룹을 동시에 선택하면 같은 기록이 각 그룹에 한 번에 등록돼요 (예: 개인 책장 + 가족 그룹)
  3. 상태 선택 — 🌱 읽을 예정 📖 독서중 🏆 완독 중 하나
    읽을 예정이면 제목·저자만으로 등록 가능, 완독이면 완료일과 별점이 필수예요
  4. 책 제목·저자 입력 후 ✨ 책정보 생성 클릭
  5. 네이버 책 검색 결과에서 내 책을 선택 → 표지·출판사·구매 링크·가격이 자동으로 채워져요
  6. 나머지 항목 입력 후 등록하기 🎉
항목설명
독서 기간시작일 ~ 완료일. 홈의 상태 버튼을 쓰면 자동 입력돼요 (5장)
평가별점 ★★★★½ — 별을 클릭, 0.5개 단위 (왼쪽 절반=반 개)
이런 사람에게 추천MBTI 여러 개 선택 가능, 💛 모두에게 옵션도 있어요
기억에 남는 문장마음에 남은 문장 필사
읽고 느낀 점자유로운 감상
💡 검색 결과가 없다고 나와요 BookKing은 제목+저자+출판사 → 제목+저자 → 제목만 순서로 자동 재검색해요. 그래도 없으면 제목·저자의 띄어쓰기와 오타를 확인해보세요 (예: 믿음사 ❌ → 민음사 ⭕). 검색에 없는 책은 직접 입력한 내용 그대로 등록해도 돼요.

6내 책장

  • 상단 메뉴 📖 내 책장 — 내가 등록한 책이 표지 그리드로 정리돼요
  • 필터: 전체 / 읽을 예정 / 독서중 / 완독
  • 정렬: 읽은 날짜 · 평가(별점) · 제목 · 저자 (한 번 더 누르면 역순)
  • 책을 클릭하면 수정·삭제·상태 변경을 할 수 있어요 (내 기록만)

7책 검색

상단 메뉴 🔍 책 검색에서 한 번의 검색으로 두 가지를 같이 보여줘요.

  1. 👥 우리 그룹의 기록 — 이 책을 누가 읽었고 별점을 몇 점 줬는지. "문장·느낀점 보기"로 그룹원의 감상을 읽을 수 있어요
  2. 🟢 네이버 책 검색 결과 — 표지·저자·가격·구매 링크. 내 책장에 등록을 누르면 책 정보가 채워진 등록 화면으로 이동해요

8랭킹

랭킹기준
📚 책 랭킹 — 우리 그룹현재 그룹에서 완독된 책. 별점 순(평균 별점) / 많이 읽은 순(완독자 수)
📚 책 랭킹 — 통합BookKing 전체 그룹 집계. 책 정보와 수치만 보이고 누가 읽었는지는 비공개
👑 독서왕우리 그룹에서 올해 완독한 권수 순. 🥇🥈🥉 메달이 붙어요
📌 내 책이 랭킹에 안 보여요 책 랭킹에는 3명 이상이 완독한 책만 올라가요. 한 사람의 별점만으로 1위가 되는 걸 막기 위해서예요. 그룹원들에게 같은 책을 권해보세요!

9운영자 가이드 운영자

그룹장이 지정한 운영자에게는 상단에 🧹 글 관리 메뉴가 생겨요.

  1. 들어가면 그룹 최신 기록 10개가 바로 보여요 → 더 보기 +10으로 계속 불러올 수 있어요
  2. 특정 글을 찾을 땐 검색 기준(사용자 이름 / 책 제목)을 골라 검색
  3. 결과에서 수정 또는 삭제 → 삭제는 한 번 더 확인을 눌러야 실행돼요
🗑 삭제해도 완전히 사라지지 않아요 삭제된 글은 목록·랭킹에서 숨겨질 뿐 데이터는 남아 있고, 누가 언제 지웠는지 기록돼요. 실수로 지웠다면 그룹장이 복구할 수 있어요.

10그룹장 가이드 그룹장

그룹장에게는 👑 그룹 관리 메뉴가 추가로 생겨요. (운영자 기능도 전부 사용 가능)

기능방법
그룹 옵션🔍 외부 검색 허용(그룹 찾기에 노출) / 👀 보기 전용(일반 그룹원은 열람만, 기록은 그룹장·운영자만) / 🙋 가입 승인제(어떤 경로로 오든 그룹장·운영자가 승인해야 가입). 모두 기본 꺼짐
가입 신청 처리승인제 그룹은 상단 🙋 가입 신청 메뉴(운영자도 가능)에서 ✅ 승인 / 거절. 대기 건수가 빨간 배지로 표시돼요
운영자 지정/해제그룹원 목록에서 운영자 지정 / 운영자 해제
그룹원 내보내기멤버 목록의 내보내기 — 내보내도 그 사람이 남긴 기록은 그룹에 남아요
초대 링크 관리남은 유효기간 확인, 복사, 새 링크 발급 (발급하면 기존 링크는 즉시 무효화)
그룹장 위임그룹원 옆 👑 위임 → 확인. 상대가 새 그룹장이 되고 나는 운영자로 전환돼요
삭제된 글 복구삭제된 글 목록에서 복구 → 책장·랭킹에 다시 반영
⚠️ 그룹장 위임은 신중하게 위임하면 되돌릴 수 없어요 — 새 그룹장이 다시 위임해줘야만 돌아올 수 있어요.

11자주 묻는 질문

Q. 회원가입이 따로 있나요?
아니요, Google 계정으로 로그인하면 끝이에요. 처음 로그인할 때 이름만 정하면 돼요.

Q. 그룹 없이 혼자 기록만 하고 싶어요.
가능해요! 가입하면 자동으로 생기는 개인 책장(『내이름의 책장』)에서 혼자 기록하면 돼요. 나중에 그룹에 가입하거나 만들면 상단 그룹 선택으로 오가며 쓸 수 있어요.

Q. 여러 그룹에 가입할 수 있나요?
네! 상단의 그룹 선택 상자로 전환해요. 기록·역할·랭킹은 그룹마다 완전히 분리돼요. 그룹에서 나가고 싶으면 홈 하단의 🚪 이 그룹에서 나가기를 누르면 돼요 (그룹장은 위임 후 가능).

Q. "보기 전용" 그룹이 뭐예요?
그룹장·운영자만 기록을 올리고 일반 그룹원은 구경만 하는 그룹이에요 (추천 도서 공지용). 내 기록을 남기고 싶다면 다른 그룹을 쓰거나 직접 그룹을 만들면 돼요.

Q. 가입 신청했는데 아직 안 들어가져요.
그 그룹은 가입 승인제예요. 그룹장·운영자가 승인하면 자동으로 그룹 목록에 추가되니 잠시 기다려주세요.

Q. 내 기록을 다른 그룹 사람이 볼 수 있나요?
아니요. 기록은 등록한 그룹 안에서만 보여요. 통합 랭킹에도 책 제목과 수치만 나가고 이름은 노출되지 않아요.

Q. 같은 책인지 어떻게 알아요?
책정보 생성으로 등록하면 ISBN으로, 직접 입력하면 제목+저자로 같은 책을 묶어서 랭킹을 집계해요. 가급적 ✨ 책정보 생성을 사용하는 걸 추천해요.

Q. 별점은 어떻게 반 개를 주나요?
별의 왼쪽 절반을 클릭하면 반 개, 오른쪽을 클릭하면 한 개예요. (0~5점, 0.5 단위)

Q. 카카오톡에서 링크를 열었더니 Google 로그인이 차단돼요 (403 오류)
카카오톡·인스타그램 같은 앱 안의 브라우저에서는 Google이 보안 정책으로 로그인을 막아요. 화면에 나타나는 🌐 기본 브라우저로 열기 버튼을 누르면 Chrome/Safari로 열려서 정상 로그인돼요. (버튼이 없으면 메뉴 ⋮ → "다른 브라우저로 열기")

Q. 그룹을 더 만들고 싶은데 "이용권이 모두 사용됐어요"라고 나와요.
그룹 생성은 기본 1개까지 무료예요. 개인 책장의 상단 메뉴 🎟️ 이용권에서 쿠폰 코드를 입력하거나 이용권 요청을 보내주세요. 운영자가 승인하면 같은 페이지에서 쿠폰을 받아 바로 쓸 수 있어요. (그룹 가입은 이용권 없이 무제한이에요)

Q. 초대 링크를 잃어버렸어요.
그룹장이 👑 그룹 관리에서 언제든 다시 복사하거나 새로 발급할 수 있어요.
